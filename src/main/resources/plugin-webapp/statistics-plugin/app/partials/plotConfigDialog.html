Plot

What can i plot here?
The start times or end times of the process/activity instances
Purpose
To identify patterns in the starting or ending times of the instances. e.g. accumulations of starting instances
What's on the x axis?
  1. no timeframe
    The start or end time of the instances. The format is here YYYY-mm-dd-hh
  2. daily
    Only the time of day of starting or ending time is viewed. Year, month and day get cut off. So in the plot it will not possible to see a difference between instances starting at the same time of day but at a different day.
  3. weekly
    Only the weekday and the time of day will be displayed. Year and month get cut off.
What's on the y axis?
All the processes/activities instances belonging to the same key will be plotted at the same y value

Options

timeframe
see the description above
clustering
To get a better impression over the data the k-means clustering algorithm can be used. The number of clusters k is an input parameter: an inappropriate choice of k may yield poor results.
  time series of duration


Plot

What can i plot here?
The time series of the durations of the chosen processes/activites
Purpose
To assess the change over time of the durations
What's on the x axis?
The start time of the instances.
What's on the y axis?
The duration of the instances

Options

show scatter plot
Shows the instances as circles
show splines
Linearly interpolates the discrete data to get a continuous plot
draw regression
Shows a linear regression line to observe a trend in the time series. This only makes sense if there really is a linear trend!

Plot

What can i plot here?
The durations of the chosen processes/activites
Purpose
To roughly assess the probability distribution of the durations by depicting the frequencies of observations occurring in certain ranges of values
What's on the x axis?
The durations of the chosen process/activity instances starting with their minimal duration and ending with their maximum. This range of durations is divided into @number_of_bins bins. The end and start values of these bins are given in percent of the range.
What's on the y axis?
The absolute frequencies of the process/activity instances

Options

number of bins
Changing the number of bins affects the width of the range of the bins. Choosing more bins results in smaller bins and thus a higher accuracy
